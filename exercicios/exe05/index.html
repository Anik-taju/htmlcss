Negrito

Para utilizar a tag em Negrito pode se usar a tag < b > mas o b e uma tag somente de forma sem sentido (nao semantica)

Mas tambem pode se usar a tag < strong > para por em Negrito essa tem forma e sentido (semantica) sendo mais eficas no uso

Italico

Para utilizar a tag em Italico pode se usar a tag < i > mas o i e uma tag somente de forma sem sentido (nao semantica)

Mas tambem pode se usar a tag < em > para por em Italico essa tem forma e sentido (semantica) sendo mais eficas no uso
